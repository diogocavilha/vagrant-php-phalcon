# Nginx / PHP7.0 / Phalcon 3.0.1 / Composer / Git / NodeJs / Npm / Bower / Gulp

# Installation
```sh
$ cd /path/project
$ git clone https://github.com/diogocavilha/vagrant-php-phalcon.git vagrant && rm -rf vagrant/.git && echo "vagrant/.vagrant" >> .gitignore
$ cd vagrant
$ vagrant up
```
